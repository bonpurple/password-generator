# リリースノート — v1.1.1

**リリース日:** 2025-12-01

## 概要

このリリースでは、より安全なパスワードを推奨するために、文字数プリセットとカスタム入力の初期値を見直しました。

---

## 変更点

- 初期の文字数プリセットを `12 / 8` から `16 / 12` に変更。
- カスタム文字数入力の既定値を `16` から `24` に変更し、長めのパスワードを生成しやすくしました。

---

## 注意事項 / マイグレーション

- 既存ユーザーでも、従来通り任意の文字数を指定できますが、UI 上の初期値が変わっています。
- 過去バージョンと**まったく同じ条件**でパスワードを再生成したい場合は、
  - 16文字プリセット → 以前の12文字相当になるように手動で調整する
  - カスタム入力 → 必要に応じて `24` 以外の値を設定する  
    など、文字数を手動で選択してください。
