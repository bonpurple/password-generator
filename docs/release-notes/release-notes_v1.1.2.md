# リリースノート — v1.1.2

**リリース日:** 2025-12-02

## 概要

このリリースでは、UI 全体のフォントスタックを見直し、日本語環境での表示差異を減らしました。  
ローカルでの利用と GitHub Pages 上での利用のどちらでも、数字と日本語の見た目が極力揃うように調整しています。

---

## 変更点

- グローバルな `font-family` を以下のように変更：
  - 変更前:
    ```css
    font-family:
      Inter,
      system-ui,
      -apple-system,
      "Hiragino Kaku Gothic ProN",
      "Noto Sans JP",
      Meiryo,
      sans-serif;
    ```
  - 変更後:
    ```css
    font-family: "Yu Gothic UI", "Noto Sans JP", Meiryo, system-ui, sans-serif;
    ```
- これにより、ローカル環境で「16文字」「12文字」などの数字だけが別フォント（例: Inter-Bold）になる現象を抑制し、  
  GitHub Pages 上の表示とのギャップを小さくしました。

---

## 注意事項

- 本リリースは **表示フォントのみの変更** であり、パスワード生成アルゴリズムや出力内容には影響しません。
- 既存ブックマークやローカルコピーは、そのまま上書きすれば自動的に新しいフォントスタックが適用されます。
